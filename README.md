## 📝 Project Summaries

### 1. Classification & Regression
* **SVM & KNN:** Implementations of Support Vector Machines and K-Nearest Neighbors including hyperparameter tuning for optimal kernel selection.
* **Naive Bayes:** Probabilistic classification models, including conflict resolution in datasets like Titanic.
* **Boosting & Ensembles:** Advanced use of Gradient Boosting, Stacking, and Voting classifiers to improve model accuracy.

### 2. Unsupervised Learning & Anomaly Detection
* **PCA:** Principal Component Analysis for dimensionality reduction and feature extraction.
* **Anomaly Detection:** Identification of rare items or events in datasets using kernel evaluation theory.
* **Clustering:** Unsupervised grouping of data points to find hidden patterns.

### 3. Time Series & Auto ML
* **Time Series:** Advanced ARIMA modeling with detailed visualizations for forecasting.
* **Auto ML:** Automated machine learning pipelines for rapid model prototyping.

### 4. Data Analysis (EDA)
* In-depth data cleaning, statistical analysis (Chi-square, Crosstabs), and visualization to derive actionable insights from raw datasets.
