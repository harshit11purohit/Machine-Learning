# Anomaly Detection: Theory and Types

Anomaly Detection is the process of identifying data points, events, or observations that deviate significantly from a dataset’s normal behavior. In many real-world scenarios, these "outliers" represent critical information such as bank fraud, structural defects, or medical issues.
## 🔍 What is an Anomaly?
An anomaly is a pattern in the data that does not conform to a well-defined notion of normal behavior. While "noise" is random error that should be removed, "anomalies" are rare but significant events that provide actionable insights.

## 🧪 Key Algorithms Implemented

| Algorithm | Type | Best For |
| :--- | :--- | :--- |
| **Isolation Forest** | Ensemble | High-dimensional data and point anomalies. |
| **DBSCAN** | Density-Based | Finding anomalies in spatial clusters. |
| **Local Outlier Factor** | Neighbors | Detecting anomalies based on local density. |
| **One-Class SVM** | Support Vector | Detecting the boundary of "normal" data. |
