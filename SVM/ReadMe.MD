# 🛡️ Support Vector Machines (SVM) Portfolio

### 1. Linear SVC (`svc.ipynb`)
Implementing a Support Vector Classifier to solve binary classification problems. This project focuses on finding the optimal hyperplane to maximize the margin between different data classes. It includes performance evaluation using confusion matrices and accuracy scores.

### 2. Non-Linear SVM Kernels (`svm_kernel.ipynb`)
Exploring advanced Kernel tricks (RBF, Polynomial, and Sigmoid) to handle non-linearly separable datasets. This notebook demonstrates how to transform low-dimensional input space into higher-dimensional feature space. It highlights the impact of $C$ and $\gamma$ hyperparameters on model boundaries.

### 3. Support Vector Regression (`svr.ipynb`)
Applying SVM principles to continuous numerical prediction rather than classification. The project utilizes an $\epsilon$-insensitive tube to find a function that fits the data points within a specific threshold. It specifically addresses how to handle outliers in regression tasks using the SVR architecture.
